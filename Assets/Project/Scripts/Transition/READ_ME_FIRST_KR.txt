DreamlandTutorialStage1 복구 패키지 (Test01)

이 패키지는 업로드된 원본 프로젝트의 Assets/DreamlandTutorialStage1 전체 폴더를 복구하고,
Stage1WaveController.cs만 Test01 수정본으로 넣은 패키지입니다.
원본 .meta 파일도 포함되어 있어 씬/컴포넌트 GUID 연결을 보존합니다.

적용 전:
1. Unity Play Mode를 종료합니다.
2. Unity Editor를 닫는 것을 권장합니다.
3. 현재 Assets/DreamlandTutorialStage1 폴더를 별도 백업합니다.

적용:
1. 이 ZIP을 프로젝트 밖의 임시 폴더에 압축 해제합니다.
2. 압축 해제된 Assets/DreamlandTutorialStage1 폴더를 프로젝트의 Assets 안으로 복사합니다.
3. 대상 폴더 전체를 교체하거나, 모든 파일을 병합/덮어쓰기합니다.
4. Assets/DreamlandTutorialStage1.meta도 함께 덮어써도 됩니다.
5. Unity를 열고 컴파일이 끝날 때까지 기다립니다.

주의:
- ZIP 안의 최상위 Assets 폴더를 macOS Finder에서 프로젝트 Assets 위에 바로 '대치'하지 마세요.
  프로젝트의 다른 Assets가 삭제될 수 있습니다.
- 반드시 DreamlandTutorialStage1 폴더 단위로 교체하세요.
