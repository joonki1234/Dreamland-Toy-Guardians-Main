# Dreamland Tutorial + Stage 1 v8

## 적용 순서

1. Unity Play 모드를 종료합니다.
2. 기존 `Assets/DreamlandTutorialStage1` 폴더를 삭제합니다.
3. 이 ZIP의 `DreamlandTutorialStage1` 폴더를 프로젝트의 `Assets` 바로 아래에 넣습니다.
4. Unity의 스크립트 컴파일이 끝날 때까지 기다립니다.
5. 상단 메뉴에서 아래 항목을 실행합니다.

`Dreamland > 튜토리얼 + Wave 1 > 스토리 + 수정가능 대사 v8 적용`

6. 완료 창이 뜨면 씬을 저장하고 Play로 확인합니다.

## 대사 수정 위치

v8 적용 메뉴를 실행하면 다음 에셋이 자동 생성되고 선택됩니다.

`Assets/GameData/Dreamland/TutorialDialogueData.asset`

Project 창에서 이 에셋을 클릭하면 Inspector에서 다음 내용을 수정할 수 있습니다.

- 미션 시작 제목과 부제
- 튜토리얼 시작 대사 목록
- 튜토리얼 적 등장 안내
- 사격, 시너지, 정화 안내 대사
- 튜토리얼 완료 대사
- Wave 사이 안내 대사
- Wave 완료 및 코어 업그레이드 대사
- 각 대사의 표시 시간

대사만 수정할 때는 적용 메뉴를 다시 실행할 필요가 없습니다. Play 모드를 종료한 상태에서 에셋을 수정한 뒤 다시 Play하면 바로 반영됩니다.

## 대사 목록 추가와 순서 변경

`튜토리얼 시작 대사` 목록의 크기를 늘리면 대사를 추가할 수 있습니다. 목록은 위에서 아래 순서대로 자동 재생됩니다.

- `Speaker`: 화자 이름
- `Message`: 실제 대사
- `Duration`: 화면에 표시되는 시간(초)

## 현재 전투 설정

- 튜토리얼 적은 플레이어 정면 약 9m에서 등장하고 이동하지 않습니다.
- Wave 적은 코어 방향으로 이동합니다.
- Wave 적 기본 이동속도는 0.35입니다.
- Wave 구성은 3마리, 3마리, 6마리입니다.
